### Instalasi

1. Kloning repositori
   ```bash
   git clone https://github.com/ekalayamerdeka/todo-app-mern.git
   ```
2. Instalasi dependensi
   cd todo-app-mern
   npm install

3. Buat File .env
   MONGO_URI=your_mongodb_connection_string
   PORT=5000

4. Jalankan Aplikasi
   npm start
